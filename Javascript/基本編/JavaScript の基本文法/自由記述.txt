【メモ】

★プロパティ
・値を参照したり設定するためのプロパティ(属性)を持つ

navigator.appName       // ブラウザの名前
navigator.appVersion    // ブラウザのバージョン

★メソッド
・動作を指定する際に用いるメソッド(処理)をサポート

window.alert("Hello");            // 警告ダイアログを表示する
window.open("xx.html", "_blank"); // 新しくウィンドウを開いて表示する

★コメントアウトの種類

//

/* この部分はコメントです。*/

/*
 * この部分がコメントです。
 */

#!

★ストリクトモード
・通常ではエラーとならないコードに対してエラーを発生させることにより、プログラミングのミスを見つけやすくする
